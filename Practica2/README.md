# React Ejemplos - Aprendiendo React desde cero

Repositorio de ejercicios prácticos para aprender React con Vite.

## Conceptos aprendidos

### Practica1 - Lista de Tareas (TODO App)
Conceptos cubiertos:
- `useState` → guardar y cambiar datos en el componente
- `props` → pasar datos del componente padre al hijo
- `.map()` → recorrer listas y mostrarlas visualmente
- `.filter()` → eliminar elementos de una lista
- `onChange` → detectar cuando el usuario escribe
- `onClick` → detectar cuando el usuario hace click
- Componentes separados → `App.jsx` y `TareaItem.jsx`
